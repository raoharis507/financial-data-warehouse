# Financial Data Warehouse for Acme Ltd

## Overview
A complete data warehouse platform for financial market data with temporal storage, REST API, analytics, and LLM integration.

## Features
- UC1: Data ingestion from multiple providers (Nasdaq, Bloomberg)
- UC2: RESTful API with 5 required queries (Q1-Q5)
- UC3: Analytics and forecasting (moving avg, price forecast, volatility)
- UC4: LLM assistant with natural language interface
